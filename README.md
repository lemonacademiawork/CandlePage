# Candle Landing Page Backend

This is the Spring Boot backend service for the Candle Making Course landing page.

## Technology Stack
- Java 21
- Spring Boot
- Maven
- MongoDB

## Project Structure
The project structure is organized by layer/component:
- `config`: Configuration classes (CORS, Security, etc.)
- `controller`: REST APIs
- `service`: Business logic definitions
- `repository`: MongoDB data access layers
- `model`: MongoDB documents and domain models
- `dto`: Data Transfer Objects for requests and responses
- `exception`: Global and custom exception handlers
- `util`: Constants and utilities
